Movie Database Project

To run:
	use `npm install` to install all dependencies
	use node server.js to start the server running
	go to http://localhost:3000/ to view the site

NPM modules / dependencies:
	express
	express-session
	pug

Folders:
	data - folder that contains all the movie, people, and user information (as well as a list of genres, and the scripts to format the data)
	node_modules - all required modules downloaded with npm
	public - all files that are served by the static server
	JS - Client Side JavaScript files for manipulating the DOM, sending AJAX requests, etc
	styles - CSS for styling HTML pages
	routes - express routers for handling /users /people and /movies urls
	views - all PUG template engine files used to generate HTML
